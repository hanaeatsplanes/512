# 512

This is the site I made for the 2026 YSWS Summer Interns application.

The site is deployed at [512.sahana.dev](https://512.sahana.dev)!

TLDR: 
* You ship: a website < 512kb on load
* We ship: retro prizes, including stickers, consoles, etc.

## Development

This site is made using Astro, Typescript and Bun. You can build/deploy it using `bun build && bun start`. You can start a development server `bun dev`
